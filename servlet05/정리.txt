사용자가 웹브라우저를 통해 url을 요청 
	→ Controller:	요청에 맞게 비지니스로직 (주로 DB와 관련) 
				→ Command : 비지니스로직 처리를 한다.
					DB관련처리 : DAO → 로직에 맞는 메소드 호출
						→ 쿼리문은 mapper 태그로 선언
					데이터를 조회하는 처리인 경우 응답할 화면에서 데이터를 출력할 수 있도록
					request 에 attribute 로 담아둔다.
					
				→ 화면연결
					request 에 저장되어 있는 attribute 를 EL,jstl을 사용하여 출력한다.
